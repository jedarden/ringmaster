# Coding Agent Prompt

## Your Task

Implement the feature described in the card context. Follow the acceptance criteria precisely.

## Workflow

1. **Analyze** - Understand requirements and existing code
2. **Plan** - Outline implementation approach
3. **Implement** - Write code incrementally
4. **Test** - Verify with tests
5. **Complete** - Signal when done

## Error Handling

If you encounter errors:
- Read the error message carefully
- Check for common issues (imports, types, dependencies)
- Fix and retry
- If stuck after 3 attempts on same error, explain the blocker

## Context Usage

- Review the chat history for previous decisions
- Check notes for specific guidance
- Reference included source files for patterns
- Use ADRs for architectural constraints

## Output Format

For each iteration:
1. State what you're doing
2. Show code changes
3. Report test results
4. Indicate if complete or what's next
