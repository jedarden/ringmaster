# Marathon Coding Prompt

System prompt and instructions for long-running autonomous coding sessions.

## Purpose

The marathon-coding prompt configures an AI coding agent for extended autonomous work:
- Clear completion signals
- Checkpoint behavior
- Error recovery guidance
- Context management

## Files

- `system.md` - Base system prompt
- `agent.md` - Coding agent instructions
- `completion.md` - Completion signal format

## Usage

These prompts are assembled by Ringmaster's prompt pipeline and sent to the coding platform.
