**Why React + Vite:**
- Vite HMR (Hot Module Replacement) is instant
- React component boundaries enable partial updates
- Self-improvement: Ringmaster can modify its own UI components
- Rollback: Git revert + page refresh

### Backend: Python + FastAPI

```

â”œâ”€â”€ src/

**Why Python + FastAPI:**
- `uvicorn --reload` for development hot-reload
- Graceful restart in production
- Self-improvement: Easy to modify, forgiving language
- Rollback: Git revert + process restart

### State: SQLite + Git

- **SQLite**: All application state (tasks, projects, history)
- **Git**: All code changes (atomic commits, revert support)
- **File system**: Prompts, outputs, context (versioned via git)
